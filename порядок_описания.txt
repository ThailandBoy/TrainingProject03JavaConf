В этом проекте будем работать через Spring core.
Конфигурационный файл сделан через JavaConfig.
Следующий проект будет работать по веб архитектуре с сервером томкат через Spring MVC.
1. Собрать проект через мавен, архетип webapp
2. в pom.xml импортируем нужные зависимости
    - spring-core; spring-beans; spring-context
3. создать директорию "resources"
    - пометить как "resources root"
    - все файлы которые мы поместим в эту папку, они будут доступны в нашем коде.
        Таково назначение данной папки ресурсов.
    - в этой папке будет размещена наша конфигурация спринга
4. создать папку "java", в ней будет весь наш java код
    - пометить папку как "sources root", так мы дадим понять "идее" что там будет храниться java код
5. Создать AppConfig.java c аннотацией
    @Configuration - данная аннотация отмечает что класс является конфигурационным
    @ComponentScan() - в этой аннотации нужно отметить базовую директорию для поиска компонентов
    @PropertySource() - в этой аннотации нужно отметить путь к файлу с данными ключ-значение, он лежит в папке resources
    Теперь в main классе указываем context
        AnnotationConfigApplicationContext context = new AnnotationConfigApplicationContext(AppConfig.class);
6. Для демонстрации примера работы, перед дальнейшей работой
   были удалены аннотации: @ComponentScan(), @Component, @Autowired
